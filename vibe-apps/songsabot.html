기능 ①

📝 사건 경위서 (학생신고서)

학생이 직접 스마트폰으로 작성하는 사건 신고 양식입니다.

🏫 현재 접속: [5-1반]
📋 사건 경위서
1. 이름
예: 김철수
2. 언제 (사건 발생 시간)
2026-04-05 오후 10:16
3. 어디서
예: 5학년 2반 교실, 운동장...
4. 무슨 일이 있었나요?
점심시간에 줄을 서 있는데 ○○이가 갑자기 뒤에서 저를 밀었습니다...
5. 나는 어떻게 행동했나요?
처음에는 하지 말라고 말했습니다...
  • 📱
    스마트폰에 최적화된 UI 학생이 쉽게 접근할 수 있도록 모바일 친화적인 양식으로 설계되었습니다.
  • 🏫
    학급 자동 식별 접속 시 현재 학급(예: 5-1반)이 자동 표시되어 별도 선택 없이 신고 가능합니다.
  • 📋
    6가지 필수 항목 구조화 이름, 시간, 장소, 상황, 나의 행동, 이유를 체계적으로 입력해 AI 분석의 정확도를 높입니다.
  • ⏱️
    즉시 처리 제출 즉시 관리자 화면에 사건이 등록되고 AI 분석이 자동 시작됩니다.
  • 🔒
    학교·학급별 접근 관리 각 학급 전용 URL로 접속해 사건이 해당 학급에 정확히 분류됩니다.
기능 ②

🤖 AI 자동 분석 (송사봇)

사건 경위서가 접수되면 AI가 즉시 분석하고, 교사가 바로 활용할 수 있는 결과물을 만들어 냅니다.

🤖
AI 분석 결과
이영희, 김민영 (통합)
1. 종합 요약
이영희 학생은 정지 상태에서 김민영 학생이 축구 활동 중 달려오다 어깨에 부딪혀 통증을 느꼈으며, 김민영 학생의 무성의한 태도에 불만을 표했습니다. 이번 사건은 우발적인 신체 접촉으로 인한 서로 다른 상황 인지 및 감정적 대응 문제로 보입니다.
📨 학부모 안내 문자 (자동 생성)
안녕하십니까, 담임교사입니다. 오늘 점심시간에 이영희 학생과 김민영 학생 사이에 교내에서 작은 충돌이 있었습니다. 학교에서는 두 학생의 이야기를 듣고 상황을 이해하며 소통 방법에 대해 지도할 예정입니다...
  • 🧠
    자동 종합 요약 생성 복수 학생의 진술을 분석해 사건의 객관적인 종합 요약을 자동으로 작성합니다.
  • ✉️
    학부모 안내 문자 자동 작성 교사가 직접 쓰지 않아도, AI가 학부모에게 보낼 정중하고 명확한 안내 문자를 즉시 완성합니다.
  • 🔄
    재분석 요청 가능 분석 결과가 만족스럽지 않으면 버튼 한 번으로 AI 재분석을 즉시 요청할 수 있습니다.
  • ⚖️
    중립적·객관적 서술 AI가 양측 진술을 균형 있게 반영하여 편향 없는 사건 요약과 안내 문자를 생성합니다.

🔗 사건 통합 분석

같은 사건에 관련된 여러 학생의 신고를 하나로 묶어 통합 분석할 수 있습니다.

📋 이영희 신고서
"정지 상태에서 김민영 학생이 달려오다 어깨에 부딪혀 통증을 느꼈습니다. 무성의한 태도에 불만을 표했습니다."
⊕
📋 김민영 신고서
"축구 중 공을 쫓다 우발적으로 발생한 가벼운 충돌이며 의도하지 않았습니다. 이영희 학생의 격한 반응에 당황했습니다."
🤖 통합 AI 분석 결과
두 학생의 진술을 종합한 결과, 이번 사건은 우발적인 신체 접촉으로 인한 서로 다른 상황 인지 차이에서 비롯된 갈등으로 판단됩니다. 양측 모두 감정적 표현을 사용했으며, 고의성은 낮은 것으로 보입니다. → 학부모 안내 문자 자동 생성 완료 ✅
