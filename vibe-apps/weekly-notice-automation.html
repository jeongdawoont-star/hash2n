✦ 왜 만들었나요
반복 업무, 이제 그만
매일 새 시트 만들고, 날짜·안전수칙·과목별 내용 입력하는 일이
한 주에 몇 번씩 반복되다 보면 정작 중요한 수업 준비 시간이 줄어들게 됩니다.
📋

알림장 시트 자동 생성

이번 주·다음 주 평일 시트를 한 번에 생성. 날짜탭·안전수칙이 자동으로 채워집니다.

📅

주간학습안내 자동 완성

DB 시트에서 원하는 주차를 클릭하면 교과별 단원·학습내용이 템플릿에 자동 입력됩니다.

⚡

Google Sheets 메뉴 연동

스프레드시트 상단에 커스텀 메뉴가 생성되어 누구나 쉽게 버튼 하나로 실행합니다.

  • 🗓️날짜 자동 계산 — 오늘 기준으로 이번 주/다음 주 평일을 자동 계산, 과거 날짜는 건너뜁니다.
  • 🛡️안전 수칙 랜덤 삽입 — 100가지 초등 안전 수칙 중 하나가 매 시트마다 무작위로 배정됩니다.
  • 📚교과목 DB 연동 — 별도 DB 시트에서 주차별 단원·학습내용을 불러와 템플릿에 자동으로 채웁니다.
  • 🔁중복 방지 — 이미 생성된 시트는 건너뛰거나 덮어쓸지 선택할 수 있습니다.
