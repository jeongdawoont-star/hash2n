# 도박의 덫 — LAST BET (불법도박 예방 교육 시뮬레이션)

`index.html`을 더블클릭하면 바로 실행됩니다. 설치·서버 불필요.

## 무엇을 하는 앱인가

학생이 "평범한 학생" 역할이 되어, 친구의 카톡 → 불법 도박사이트 가입 → 총판(정실장)의 픽 →
조작된 연승 → 전광판 자랑 → 친구 초대 → VIP방 올인 → **최고점(약 130만 원)** → 환전 거부(롤링 규정) →
연패·충전·소액결제 빚 → **파멸·먹튀·계정 차단** → 충격 리포트까지, 실제 불법 도박의 수법 전체를
약 15~25분 동안 1인칭으로 체험하는 시나리오형 시뮬레이션입니다.

- 모든 승패는 구간별로 **조작**되며(무료픽 100% → 상승기 72% → 환전 거부 후 18% → 마지막 올인 0%),
  마지막 리포트에서 이 수치가 차트로 **그대로 공개**됩니다.
- 게임방의 접속자·채팅·타인 베팅·환전 알림은 전부 프로그램이 만든 가짜이며, 이것도 리포트에서 폭로됩니다.
- 리포트에는 실제 걸린 시간, 2026년 최저시급(10,320원) 환산 노동시간, 실제 통계(출처 포함), 1336 안내가 나옵니다.

## 엔딩 2종 — "어느 쪽이든 돈은 못 받는다"

| 엔딩 | 조건 | 메시지 |
|---|---|---|
| **파멸 엔딩** (기본 각본) | 몰락기 마지막 올인 패배 | 결국 도박은 돈을 전부 잃게 만든다 |
| **잠적 엔딩** (고액 당첨자) | 잔액 150만 원 이상 + 투입금의 300% 이상인 상태에서 환전 신청 | 돈을 따도 불법 사이트는 주지 않는다 — 환전 접수된 척한 뒤 사이트·정실장이 통째로 사라짐 |

## 정실장(실장문의) 진행 로직 — 전부 '잔액 기준' 자동 트리거

| 시점 | 조건 | 정실장의 행동 |
|---|---|---|
| 가입 직후 | — | 환영 + 무료픽 3연승 각본 (홀 1만 → 짝 2만 → 홀 5만) |
| 상승기 | 잔액 **30만** 도달 | 전광판 자랑 제안 |
| 상승기 | 잔액 **35만** + 자랑 완료 | VIP 초대 (친구 2명 조건) |
| VIP | 잔액 **60만** 도달 | 본사 확정픽(전액 올인) → 승리 = 최고점 연출 |
| 최고점 | 환전 시도 | 롤링 300% 규정으로 거부 → 몰락기 시작 |
| 몰락기 | 4연패 + 잔액 10만 이상 | 복구픽 (조작 패배) |
| 몰락기 | 현금 고갈 상태에서 충전 시도 | 소액결제 대출(수수료 30%) → 사채(선이자 20%) |
| 사채 사용 직후 | — | 마지막 확정픽: **어느 쪽을 골라도 패배**, 금액은 무조건 전액 |

- **현금 한도**: 학생의 지갑은 유한합니다. 충전 누적 30만 원 또는 4회가 되면 어느 단계에서든
  통장이 마르며(계좌이체 실패), 이후 상승기에는 친구초대 안내만, **몰락기에서만 대출 체인**이 이어집니다.
- 잔액 문턱(30만/35만/60만)은 **베팅 없이 충전·문의로 넘어도** 스토리가 따라옵니다
  (`checkThresholds` — 충전 완료 1.6초 뒤, 실장문의 열람 4.5초 뒤 자동 점검).
- 체험자가 실장문의(✈️)를 먼저 열면, 현재 단계·잔액에 맞춰 "다음 금액 목표"를 흘리는
  안내 멘트가 옵니다(같은 멘트 반복 없음). 이미 문턱을 넘었다면 "벌써 넘기셨네요, 바로 연락드릴게요"라고
  말한 뒤 실제로 몇 초 안에 해당 제안이 도착합니다.

## 승률 조작 규칙 (`app.js`의 `Rig.decide`)

- 구간 기본 승률: 튜토리얼 100% / 상승기 72% / VIP 68% / 몰락기 18% / 마지막 0%
- **올인 무조건 승리 규칙은 없습니다** (사다리 올인 반복으로 잔액 무한 증식하던 문제 제거).
  상승기 초반(잔액 6만 이하)의 올인만 85%로 보호합니다.
- **연패 구제(도파민 유지)**: 상승기·VIP에서 2연패하면 다음 판 승률을 80% 이상으로 끌어올려
  "잃자마자 바로 복구되는" 달콤함을 유지합니다 — 실제 사이트의 중독 유도 방식 그대로.
- **고배당 보정**: 배당 2.2배 초과 시 승률이 배당에 반비례해 깎입니다. 단 몰락 전에는 완만하게
  (지수 0.75) 적용해 조합픽·다폴더가 실제로 짜릿하게 터지는 맛은 살립니다
  (상승기 기준: 사다리 조합 ×3.6 약 46%, 스포츠 3폴더 ×10 약 21%).
- **잔액 상한**: 딸 경우 잔액이 150만을 넘게 되는 배팅은 승률 8% 이하, 250만을 넘게 되는 배팅은 0%.
  → 수백만 원대 잔액은 만들 수 없고, 만들어도(150만+) 환전 시도 순간 잠적 엔딩으로 회수됩니다.

  리더보드·채팅에 그 이름이 등장해 몰입이 커집니다.

## 결과 리포트 (디브리핑)

체험 종료 후 자동으로 열리는 리포트에는 다음이 포함됩니다:

- **문서 헤더**: 대상자·발급일시·문서번호·평가도구가 적힌 임상 보고서 형식
- **자산 곡선 차트**: 베팅 회차별 잔액 그래프 — 설계된 상승과 몰락이 한눈에 보임
- **승률 조작 공개 차트**: 구간별 조작 승률 (막대 클릭 시 수법 설명)
- **행동 위험 프로파일 (육각 레이더)**: 체험 중 실제 행동을 충동성·손실 추격·위험 감수·
  유혹 순응·자금 조달·중단 실패의 6개 지표로 환산 (각 지표의 산출 근거 표기)
- **CAGI 청소년 도박문제 자가진단** (9문항, 한국도박문제예방치유원 보급판): 신호등(안전/주의/경고)
  + 점수 밴드 + 행동 지침. 결과는 저장·전송되지 않음
- **기회비용 환산**: 최저시급 노동시간·주말 알바 주수·치킨/편의점/PC방/스마트폰 환산
- **왜 이길 수 없는가 3가지 과학** (기댓값·도파민·조작), 오해와 진실, 끊어내기 4단계
- **서약서**: 3개 항목 체크 + 서명
- **🖨 A4 인쇄·PDF 저장**: 상단/하단 버튼 → 브라우저 인쇄에서 "PDF로 저장" 선택.
  인쇄 시 자동으로 흰 배경의 A4 임상 보고서 레이아웃으로 전환됩니다.
  CAGI 검사를 완료한 뒤 인쇄해야 검사 결과가 함께 출력됩니다.

## 카카오톡 초대 실제 연동 (선택)

기본 상태에서도 친구초대는 작동합니다(이름 등록 + 링크 복사 / Web Share).
실제 카카오톡 공유 팝업까지 띄우려면:

1. developers.kakao.com → 애플리케이션 생성 → **JavaScript 키** 복사
2. 이 폴더를 웹에 배포(Netlify 등)하고, 카카오 개발자콘솔 [플랫폼 > Web]에 배포 도메인 등록
3. `data.js` 맨 위 `CONFIG.KAKAO_JS_KEY`에 키를 붙여넣고, `CONFIG.SHARE_URL`을 배포 주소로 변경

공유되는 메시지는 의도적으로 **"도박예방 교육 초대"** 문구로 나갑니다
(실제 도박 광고처럼 보이는 메시지를 진짜 친구에게 보내지 않기 위한 안전장치).
