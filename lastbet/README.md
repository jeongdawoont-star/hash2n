# 도박의 덫 — LAST BET (불법도박 예방 교육 시뮬레이션)

`index.html`을 더블클릭하면 바로 실행됩니다. 설치·서버 불필요.

## 무엇을 하는 앱인가

학생이 "평범한 학생" 역할이 되어, 친구의 카톡 → 불법 도박사이트 가입 → 총판(정실장)의 픽 →
조작된 연승 → 전광판 자랑 → 친구 초대 → VIP방 올인 → **최고점(약 130만 원)** → 환전 거부(롤링 규정) →
연패·충전·소액결제 빚 → **파멸·먹튀·계정 차단** → 충격 리포트까지, 실제 불법 도박의 수법 전체를
약 15~25분 동안 1인칭으로 체험하는 시나리오형 시뮬레이션입니다.

- 모든 승패는 구간별로 **조작**되며(무료픽 100% → 상승기 72% → 환전 거부 후 18% → 마지막 올인 0%),
  마지막 리포트에서 이 수치가 차트로 **그대로 공개**됩니다.
- 게임방의 접속자·채팅·타인 베팅·환전 알림은 전부 프로그램이 만든 가짜이며, 이것도 리포트에서 폭로됩니다.
- 리포트에는 실제 걸린 시간, 2026년 최저시급(10,320원) 환산 노동시간, 실제 통계(출처 포함), 1336 안내가 나옵니다.

## 수업 활용 팁

- 우측 상단(모바일은 하단) **"결과 바로 보기"** 버튼으로 언제든 체험을 끊고 리포트로 이동할 수 있습니다.
- 좌측 상단 🔊 버튼으로 효과음을 끌 수 있습니다.
- 닉네임은 실명 대신 별명 입력을 권장하세요. "친구 초대" 단계에서는 옆 친구의 별명을 넣게 하면
  리더보드·채팅에 그 이름이 등장해 몰입이 커집니다.

## 카카오톡 초대 실제 연동 (선택)

기본 상태에서도 친구초대는 작동합니다(이름 등록 + 링크 복사 / Web Share).
실제 카카오톡 공유 팝업까지 띄우려면:

1. developers.kakao.com → 애플리케이션 생성 → **JavaScript 키** 복사
2. 이 폴더를 웹에 배포(Netlify 등)하고, 카카오 개발자콘솔 [플랫폼 > Web]에 배포 도메인 등록
3. `data.js` 맨 위 `CONFIG.KAKAO_JS_KEY`에 키를 붙여넣고, `CONFIG.SHARE_URL`을 배포 주소로 변경

공유되는 메시지는 의도적으로 **"도박예방 교육 초대"** 문구로 나갑니다
(실제 도박 광고처럼 보이는 메시지를 진짜 친구에게 보내지 않기 위한 안전장치).

## 파일 구성

| 파일 | 역할 |
|---|---|
| `index.html` | 화면 뼈대 (폰 프레임, 사이트, 카톡/텔레그램 UI) |
| `data.js` | 설정(CONFIG)·시나리오 대사·가짜 유저·통계 데이터 |
| `sports-data.js` | 스포츠 토토 실경기 DB (betedu에서 그대로 추출: 축구·야구·농구·배구 각 12경기) |
| `app.js` | 엔진: 상태·사운드·UI·채팅·초대·충환전·시나리오 디렉터 |
| `games.js` | 게임 5종: 황금성 슬롯 / 파워볼 홀짝 / 사다리 / 스포츠 토토(조합배팅) / VIP 바카라 |
| `debrief.js` | 결과 리포트(스탯 타일·승률 조작 차트·다짐) |
| `style.css` | 전체 스타일 |
